# Practica_2
## Como abrir los programa y utilizarlos
1. Primero de todo, para utilizar el convertidor de decimal a binario, deberás acceder al **repositorio** y abrir el archivo correspondiente
	1. Una vez abierto el archivo, te pedirá que le des un *número* para pasarlo de sistema **decimal a binario**
	2. Tras definir el número para pasarlo a binario, te pedirá con cuantos digitos querrás mostrar dicho número en binario
	3. Por último, te mostrará el número que le has dado en binario con la cantidad de digitos que le habías pedido
2. Si quieres utilizar el convertidor de binario a decimal es mucho mas sencillo, deberás acceder al **repositorio** y abrir el archivo correspondiente
	1. Una vez abierto el archivo, te pedirá que le des un *número* en binario para pasarlo de sistema **binario a decimal**
	2. Por último, te mostrará el número en decimal que querías convertir desde binario 
3. Pero si quieres utilizar el convertidor de binario a hexadecimal también es mas sencillo, deberás acceder al **repositorio** y abrir el archivo correspondiente
	1. Una vez abierto el archivo, te pedirá que le des un *número* en binario para pasarlo de sistema **binario a hexadecimal**
	2. Por último, te mostrará el número en hexadecimal que querías convertir desde binario
